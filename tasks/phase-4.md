# Phase 4 Tasks

Goal: Add kitty theme and terminal transparency.

## Tasks

1. Create march kitty config files.
2. Set terminal font to JetBrainsMono Nerd Font.
3. Set opacity target to `0.72`.
4. Add the fallback color palette from `docs/theme.md`.
5. Create a simple manual theme picker plan before implementing automation.
6. Keep generated theme files separate from handwritten config.
7. Add a command/script to open kitty with the march config.

## Done When

- Kitty can load the march config.
- Terminal text remains readable on the current wallpaper.
- Transparency is visible but not distracting.
- The theme can be changed without depending on End-4 helpers.
