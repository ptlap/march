# macrch UI Theme

Name: macrch Glass Hybrid

## Style

- macOS-inspired.
- Arch identity.
- Glassmorphism.
- Blur.
- Rounded corners.
- Keyboard-first.
- No dock, only top bar.

## Bar

- Three separated islands: left, center, right.
- Height: 32px.
- Radius: 16px.
- Floating top bar.
- Translucent dark glass background.
- Thin white transparent border.
