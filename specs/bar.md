# macrch Bar Spec

Technology: Quickshell/QML

## Layout

Left island:

- Arch-like logo.
- Workspace numbers: 1 2 3 4 5.
- If workspace has an app, show an app icon.
- Active window title.

Center island:

- Clock.
- Music info.

Right island:

- Tray.
- Wi-Fi.
- Bluetooth.
- Volume.
- Brightness.
- Battery.
- CPU/RAM.
- GPU mode.
- Power profile.

## Behavior

- Top floating bar.
- No dock.
- Smooth hover animation.
- Active workspace highlighted.
- Glass blur background.
