# march Launcher Spec

Launcher: rofi-wayland, with fuzzel fallback until rofi-wayland is installed.

## Style

- Spotlight + command palette hybrid.
- Opened with Super + Space.
- Centered on screen.
- Rounded.
- Translucent.
- Keyboard-first.

## Use Cases

- Launch apps.
- Run commands.
- Change wallpaper.
- Switch theme.
- Trigger scripts.
